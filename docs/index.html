HumanTag SDK v1.3.2

Biometric Watermarking & Cryptographic Media Verification SDK

Apache 2.0 Licensed · Patent Pending

Overview

HumanTag is a modular SDK for real-time biometric authentication and media integrity. It embeds cryptographically signed biometric watermarks directly into digital content (video, audio, or metadata) to verify that a real human was present at the moment of recording.

Features

  • Biometric signal analysis: PPG, GSR, EKG
  • Entropy scoring and spoof resistance
  • Secure signature and payload encryption
  • Watermark embedding and public-key verification
  • CLI tools, SDK modules, and JSON output
