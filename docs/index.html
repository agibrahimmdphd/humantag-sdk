What is HumanTag?

HumanTag ensures authenticated human presence in recorded media using real-time biometric entropy and cryptographic watermarking.

Why It Matters

As generative AI advances, HumanTag provides a verifiable way to confirm that media was captured with a real, live human — not a simulation.

Key Features

  • Entropy-verified biometric signals (PPG, GSR, EKG)
  • Cryptographically signed payloads (ECDSA)
  • Watermark embedding & verification logic
  • SDK modules for signing, spoof detection, entropy scoring
