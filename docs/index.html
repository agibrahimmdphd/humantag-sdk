🔍 Overview

HumanTag is a modular SDK that authenticates real-time human presence in recorded media using biometric entropy, cryptographic signing, and watermark embedding. Designed for wearables, mobile, embedded, and cloud devices.

💡 Key Features

  • Entropy-verified biometric signals (PPG, GSR, EKG)
  • Cryptographically signed payloads (ECDSA)
  • Watermark embedding & verification logic
  • SDK modules for signing, spoof detection, entropy scoring
