Overview

HumanTag provides real-time biometric authentication and cryptographic watermarking to verify the presence of a real human during digital media capture.

Key Features

  • Biometric entropy simulation (PPG, GSR, EKG)
  • Secure cryptographic signature generation (ECDSA)
  • Watermark payload creation and verification
  • Forgery detection using entropy thresholds
