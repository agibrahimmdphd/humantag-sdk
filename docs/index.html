About HumanTag

HumanTag is a cryptographically verifiable biometric watermarking SDK that enables real-time authentication of human presence in recorded or streamed digital media. It is designed for scalable integration into video platforms, medical devices, mobile applications, and secure content verification pipelines.

Key Features

  • Entropy-verified biometric signals (PPG, GSR, EKG)
  • ECDSA-based digital signatures for payload integrity
  • Watermark embedding and cryptographic replay prevention
  • Spoof detection using entropy scoring and threshold gating
  • Designed for hardware, mobile, and embedded systems
