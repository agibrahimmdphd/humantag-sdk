What is HumanTag?

HumanTag is an SDK for cryptographically verifiable biometric watermarks using real-time physiological signals. It provides strong guarantees that media was generated with a real human — not AI.

Key Features

  • Entropy-verified biometric signals (PPG, GSR, EKG)
  • Cryptographically signed payloads (ECDSA)
  • Watermark embedding & verification logic
  • Signature spoof detection, entropy scoring
